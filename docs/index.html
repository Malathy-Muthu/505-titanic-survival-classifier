Titanic Survival Classifier Prediction Improvement

Steps taken to improve model precision/accuracy/AUC:
1) Included the following features by creating dummies in pandas, [ 'female', 'male', 'Cabin Class 1', 'Cabin Class 2', 'Cabin Class 3', 'Cherbourg', 'Queenstown', 'Southampton', 'sib0', 'sib1', 'sib2', 'sib3', 'sib4', 'sib5', 'Age_1_20', 'Age_20_28', 'Age_28_38', 'Age_38_80', 'Mr.', 'Mrs.', 'Miss', 'VIP']
2) Over sampling for the output values of 0 to create balanced dataset.
3) Tried the models ['naive bayes', 'logistic regression', 'k-nearest neighbors', 'random forest','svc','xgboost','decision tree','ksvm','Bagging classifier','gradient boost','adaboost','ridge classifier','extra tree ensemble']
Bagging classifier of decision tree gave the highest accuracy 81.7 (ROC curve 0.812, Precision of 0.829)
